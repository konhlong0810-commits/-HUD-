# 恐龙HUD - Dino HUD

一个将手机传感器数据实时桥接至网页端的抬头显示器（HUD）Android 应用，适用于车辆驾驶、无人机操控等场景的数据可视化。

## 简介

恐龙HUD（Dino HUD）是一款轻量级 Android 原生应用，通过内置 WebView 加载远程 HUD 页面，并将手机内置的加速度计、陀螺仪、气压计及 GPS 定位数据实时回调至网页端 JavaScript。应用本身不渲染任何 UI——所有界面由远程网页自定义，实现完全灵活的抬头显示效果。

HUD 页面地址：`https://konglong-milk.top:4002/hud`

## 主要功能

- **传感器桥接**：将 Android 原生传感器数据（加速度计、陀螺仪、气压计）实时注入 WebView 的 JS 上下文
- **GPS 定位回调**：高精度 GPS 数据（经纬度、海拔、精度、速度）持续推送至网页端
- **全屏 WebView**：隐藏标题栏，沉浸式全屏显示
- **自动恢复**：应用切后台时自动暂停传感器采集，回到前台自动恢复，节省电量
- **按需启动**：网页端调用 `AndroidBridge.ready()` 后才开始传感器数据流，避免空跑
- **返回键接管**：优先响应 WebView 内网页返回，再退出应用
- **自定义 UA**：WebView 请求携带 `DinoHUD/0.0.1` 标识，便于服务端识别

## 传感器数据格式

应用通过 JS 回调函数 `onAndroidData(json)` 向网页端推送以下 JSON 数据：

| 类型 | JSON 示例 |
|------|-----------|
| **加速度计** | `{"t":"accel","x":0.123,"y":9.801,"z":1.234}` |
| **陀螺仪** | `{"t":"gyro","x":0.0123,"y":-0.0456,"z":0.0078}` |
| **气压计** | `{"t":"pressure","hPa":1013.25}` |
| **GPS** | `{"t":"gps","lat":31.23041600,"lng":121.47370100,"alt":4.5,"acc":3.0,"spd":0.00}` |

## 注意事项

- 本应用请求**高精度 GPS 权限**，首次启动需用户授权，否则仅能获取传感器数据
- 清除流量模式（`usesCleartextTraffic=true`）已开启，支持 HTTP 加载，生产环境建议切换为 HTTPS
- 远程 HUD 页面需实现 `onAndroidData(json)` 函数才能接收数据；同时可调用 `AndroidBridge.ready()` 告知原生层开始采集
- 传感器采集频率为 `SENSOR_DELAY_GAME`（约 20ms/次），高频场景下请注意网页端性能
- 应用默认全屏无状态栏，若需调试可修改 `AndroidManifest.xml` 中的 theme 配置
